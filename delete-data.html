Delete Your Data – Rankiya

If you would like to request deletion of your data from Rankiya, please follow the steps below:

Steps to Request Data Deletion

  • Send an email to: support@rankiya.com
  • Use subject: "Delete My Data"
  • Include your name and class details

Data That Will Be Deleted

  • Name and profile information
  • Quiz attempts and scores
  • Ranking data
  • App usage activity

Data Retention

Some anonymized data may be retained for analytics and security purposes.

Processing Time

Your request will be processed within 7 working days.
